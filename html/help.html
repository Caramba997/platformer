Dieses Spiel ist ein simpler Platformer basierend auf bekannten Mario-Titeln.

Erwarte unfertige Bereiche und Fehler, da aktuell noch kein weitreichendes Testen durchgeführt wurde.


Steuerung (Keyboard)

Nach links bewegen: [a] oder [Pfeil nach links]

Nach rechts bewegen: [d] oder [Pfeil nach rechts]

Springen: [w] oder [Leertaste] oder [Pfeil nach oben]

Sprinten: [Shift]

Schießen: [f] oder [Enter]


Steuerung (Touch)

Auf Mobilgeräten kann der Charakter mit Berührung der markierten Bereiche gesteuert werden. Das Spiel ist aktuell noch nicht für Mobilgeräte optimiert und funktioniert dort nur sehr oberflächlich. Der Editor sollte mobil am Besten nicht verwendet werden.


Ton

Du kannst den Ton im Hauptmenü an- und ausschalten.


Profil

Du kannst dir mit Nutzernamen und Passwort einen Account erstellen, um selbst im Editor Level zu erstellen oder die Level von anderen Nutzern zu spielen.

Im Profil-Bereich kannst du deine Account-Informationen sehen und deine selbst erstellten Level verwalten.

Um Level von anderen Nutzern zu abonnieren oder deabonnieren, lasse dir über das Hautmenü über den entsprechenden Button alle vorhandenen Levels anzeigen. Abonnierte Levels findest du stets zum Spielen im Hauptmenü.


Fortschritt

Wenn du ein Level abschließt, wird der Fortschritt automatisch in deinem Profil gespeichert. Über das Hauptmenü kannst du auch zu den Bestenlisten gelangen und deine Rekorde mit anderen vergleichen.


Editor

Der Editor ist eine umfangreiche grafische Oberfläche, in der du selbst Level erstellen und bearbeiten kannst.

Die Nutzung wird anfangs etwas schwer fallen, eine umfangreiche Dokumentation aller Funktionen existiert zur Zeit noch nicht aber wird möglicherweise in einer späteren Version dazukommen.

Um den Einstieg zu erleichtern, findest du hier eine Kurzanleitung für die grundlegenden Funktionen:

Wenn du das erste Mal den Editor besuchst, wirst du gefragt, ob du ein Level bearbeiten oder ein neues erstellen möchtest. Solange du kein Admin bist, kannst du an diesem Punkt nur ein neues erstellen.

Nach einem Klick auf [Neu] siehst du einen leeren Entwurf mit ein paar grundlegenden Standardeinstellungen.


In der oberen Menüleiste kannst du zurück zum Hauptmenü gelangen, ein anderes Level laden, dein Level speichern und die gespeicherte Version des Levels spielen. Außerdem kannst du hier jegliche neue Spielobjekte erstellen.

In der linken Leiste, die du auch einklappen kannst, siehst du eine Übersicht über alle vorhandenen Spielobjekte sowie einige Welt-Einstellungen.

In der rechten Leiste siehst du stets die einzelnen Eigenschaften zum ausgewählten Spielobjekt.

In den Welteinstellungen kannst du im Abschnitt Meta deinem Level einen Namen geben, die zeit in Millisekunden festlegen, einen Hintergrund und eine Hintergrundmusik auswählen, die Dimensionen des Levels festlegen sowie einstellen, ob sich der Hintergrund vertikal bewegen soll (Deaktiviere diese Einstellung falls dein Level sehr flach ist).

Im Abschnitt Spieler kannst du die Startposition sowie den Startzustand des Spielcharakters festlegen.

Im Abschnitt Ziel kannst du die Position der Zielflagge festlegen. Achtung: Zusätzlich zur Zielflagge gibt es standardmäßig eine Art kleinen Turm, auf der die Flagge steht, der unter Objekte zu finden ist und bestenfalls auch bewegt werden sollte.


In der Mitte siehst du dein Level, wie es nachher auch beim Spielen aussieht. Der Unterschied ist, dass ausgewählte Spielobjekte rot markiert sind und du einen roten Cursor hast, den du mit den Pfeiltasten bewegen kannst.

Du kannst die Cursor-Position auch manuell durch Eingabe von Koordinaten setzen. Das Häkchen daneben sorgt sofern gesetzt dafür, dass bei Auswahl eines Spielobjekts der Cursor nicht seine Position ändert.

Du kannst Spielobjekte einerseits über die Übersicht in der linken Seite auswählen, andererseits kannst du auch auf das Objekt im Spiel direkt klicken.


Jedes Spielobjekt hat unterschiedliche Eigenschaften. Manche Objekte haben komplexere Eigenschaften, die nicht unbedingt selbsterklärend sind. Beispielweise können Gegner auf sehr unterschiedliche Weisen agieren. Sie können statisch sein, zwischen zwei Punkten hin und herfliegen, auf Böden laufen uvm.

Außerdem können sie unbesiegbar sein, durch Sprünge auf den Kopf getötet werden oder auch nur durch Schüsse. Probiere einfach aus, welche Einstellung welchen Effekt im Spiel hat.


Wenn du fertig bist, kannst du dein Level speichern. Wenn du im Speicherdialog bestätigst, wird dein Level automatisch auf dem Server gespeichert und ist ab sofort zum Spielen verfügbar.

Du kannst jederzeit an deinem Level weiterarbeiten und neue Versionen speichern. Dabei wird stets die alte Version überschrieben.


Technologie

Das Projekt basiert auf den grundlegenden Web-Technologien HTML, CSS, und JavaScript. Im Backend steht ein NodeJS-Server angebunden an eine MongoDB-Datenbank.

This game is a simple platformer based on Super Mario games.

Expect unfinished areas and bugs, it is still in early development and requires some more testing.


Controls (Keyboard)

Move left: [a] or [Arrow left]

Move right: [d] or [Arrow right]

Jump: [w] or [Space] or [Arrow up]

Run: [Shift]

Shoot: [f] or [Enter]


Controls (Touch)

On mobile devices, the character can be controlled by touching the corresponding areas. The game is currently not optimized for mobile devices and is not well playable there. The editor should only be used on a computer or bigger device.


Sound

You can turn the sound on and off in the main menu.


Profile

You can create an account using an username and password, to create your own levels in the editor and play other users levels.

In your profile you can see your account information and your created levels.

To subscribe to levels from other users, you can browse all levels using the button on the main menu page. Subscribed levels are always shown in the main menu afterwards.


Fortschritt

If you complete a level, your progress is automatically saved in your profile. You can compare yourself with other players in the highscore lists.


Editor

The editor is an interactive graphical interface to allow creating and modifying your own levels.

It may be hard to use in the first instance, a detailed documentation may come in the future.

Here is a short tutorial for the basic functionalities:

When you open the editor for the first time, you are asked if you want to load or create a level. As an admin, you can also load the standard levels.

After clicking [New], you get an empty level with some basic configurations.


In the top menu bar, you can return to main menu, load another level, save your level and play the saved version of the level. Additionally, you can create any new game object here.

In the left bar, you see an overview of all existing game objects aswell as some global world configuration options.

In the right bar, you see the properties of the currently selected game object.

In the world section, you can define some meta information about the level like name, time in milliseconds, background, music, dimensions and if you want a vertical parallax (deactivate if the vertical size of the level is small).

In the player section, you can set the start position and state of the character.

In the finish section, you can set the position of the finish flag. Hint: In addition to the flag, there is a little tower where the flag should be placed on, which can be found in the objects section and should be moved respectively.


In the middle you see the game view, which shows how your level will look while playing. Selected objects are marked red. You have a cursor that is a little red box and can be moved with arrow keys.

You can also move the cursor manually by entering coordinates. The checkbox defines if the cursor should jump to newly selected objects.

You can select game objects from the left bar or by just clicking on them in the game view.


Every game object type has different properties. For example, enemies can behave very different. They can be static, fly between two points, stay on a ground etc.

They can also be invincible or be killed by jump or shot. Just try which property setting has which effect in the game.


If you are done, you should save your level. If you confirm in the save dialog, your game is ready to play.

You can always continue working on your level and overwrite the current version.


Technology

This project is based on the core web technologies HTML, CSS and JavaScript. The backend is modelled as a NodeJS server using a MongoDB database.
